## 🛡️ ASR Configurator

Live Demo: 🌐  https://asrconfigugrator.pages.dev , https://asrconfig.pages.dev , https://asrconfigurator.pages.dev

ASR Configurator is a modern, interactive web tool designed to help users understand, configure, and deploy Microsoft Defender Attack Surface Reduction (ASR) rules with confidence.

This project focuses on security awareness, clarity, and safe configuration, making advanced Windows security controls accessible to both beginners and professionals.
## 🚀 Features

🔐 Full ASR Rule Coverage
Browse and configure all major Microsoft Defender ASR rules with clear explanations and examples.

- ⚖️ Security Presets

    - Disabled (All Off)

    - Basic Protection

    - Balanced (Recommended)
 
    - Strict Protection

- Developer-Friendly

- 🎛️ Per-Rule Control

    - Block

    - Audit

    - Disabled

- 📜 Instant Code Generation

     - PowerShell (local machines)

     - Group Policy (GPO)

    - Microsoft Intune (JSON)

- 📱 Responsive UI

Optimized for desktop, tablet, and mobile devices

Touch-friendly controls and adaptive layout

- 🌗 Light & Dark Mode

Theme preference saved locally

- 📋 One-Click Copy

Copy generated configuration directly to clipboard
## 🧠 Why ASR Matters

Attack Surface Reduction rules help prevent:

Macro-based malware

- Credential theft

- Ransomware behavior

- Script-based attacks

- Living-off-the-land techniques (LOLBins)

Misconfiguration can break applications — this tool emphasizes education first, audit before block, and safe defaults.
## 🧩 How It Works

- Choose a Preset
Start with a predefined security posture.

- Customize Rules
Select individual ASR rules and switch between Block, Audit, or Disabled.

- Generate Code
Pick your deployment method:

- PowerShell

- Group Policy

- Intune

Apply Carefully
Always test in Audit mode before enabling Block in production.
